About

The Fisherman's Atlas is a web-based application dedicated to help fly fishermen in Pennsylvania and the country. This application provides for essential information such as stream flowrate, temperature, fishing season, and stream pollution.

This application draws real-time stream data from USGS WaterWatch Streamflow service: https://waterservices.usgs.gov and fishing season info from Pennsylvania Fish and Boat Commission: https://www.fishandboat.com/Fish/FishingRegulations/Pages/OpeningDaysDetermined.aspx

This application is a class project for GEOG 461 in the Department of Geography of the Pennsylvania State University during the fall 2018 semester. The development of this application is supervised by Prof. Alan M. MacEachren and Arif Masrur.

This application is developed by Brittany Waltemate, Han Yu, Yu Zhong, all of whom are undergraduate students of the Penn State University.

Disclaimer: This Fisherman's Atlas application is only an undergraduate class project and is not intended to be used as official reference and/or information source of stream condition. Under no circumstances should this application be used for safety information.
